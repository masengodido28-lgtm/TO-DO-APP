To-Do App

A simple and functional To-Do application built using HTML, CSS, and Vanilla JavaScript.
This project allows users to add tasks, mark tasks as completed, delete tasks, and filter tasks based on their status.

Features
Add new tasks
Mark tasks as completed
Delete tasks
Filter tasks by:
All
Active
Completed
Data persistence using localStorage
Simple and clean user interface
Built With
HTML5
CSS3
Vanilla JavaScript
Purpose of the Project

The purpose of this project was to practice JavaScript fundamentals and improve my understanding of DOM manipulation, event handling, arrays, functions, and localStorage.

This project also helped me understand how different parts of a web application work together to create an interactive user experience.

Challenges Faced

One challenge I faced was making sure tasks stayed on the page even after refreshing the browser. I learned how to use localStorage to save and retrieve data.

Another challenge was filtering tasks correctly between active and completed tasks. I also struggled with connecting JavaScript functions to the HTML elements at first, but with practice I became more comfortable working with events and the DOM.

What I Learned

Through this project I learned:

How to manipulate the DOM using JavaScript
How to use event listeners
How to work with arrays and objects
How to create dynamic HTML elements
How to use localStorage for data persistence
How to filter and update displayed content
MDN Documentation Reflection

The MDN page that helped me the most was the localStorage documentation. It explained how browsers can store data even after the page reloads. This was useful because I wanted the tasks to stay saved after refreshing the app. The examples on MDN were simple and easy to understand, which helped me connect JavaScript with real project features. I also learned more about event listeners and DOM manipulation from MDN. Using the documentation helped me improve my problem-solving skills and understand how developers use official resources when building projects.
